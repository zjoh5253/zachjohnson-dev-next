WorkSumo is a freelance marketplace designed to reduce the friction between businesses looking for skilled help and professionals seeking meaningful project work. The platform focuses on quality matching over volume — surfacing the right candidates rather than overwhelming clients with noise.

## What I Built

The platform includes a full client-facing project posting flow, a professional profile system with portfolio galleries, and a real-time messaging layer built on Firebase. Authentication, file storage, and serverless functions all run on Firebase infrastructure, keeping operational overhead minimal.

The frontend is built in React with TypeScript, styled for clarity and speed. Search and filtering use a combination of Firestore queries and client-side logic to keep response times fast without a dedicated search service.

## Highlights

- Real-time project chat between clients and freelancers
- Skill-based filtering and discovery
- Portfolio image uploads with preview
- Mobile-responsive layout throughout
- Press coverage for the platform's approach to talent matching
